Terraform S3 Bucket Project

This project sets up a basic S3 bucket using Terraform.
It’s a small, clean example meant to show how to structure Terraform code properly and keep things readable.

What this project does
Creates an S3 bucket

Enables versioning

Uses variables for region and bucket name

Keeps the Terraform layout simple and easy to follow

Files in this repo
main.tf — S3 bucket + versioning

variables.tf — input variables

outputs.tf — output values

providers.tf — AWS provider setup
